# 摄影爱好者手机网页

一个专门为手机浏览优化的摄影爱好者网站，包含作品展示、摄影教程、器材推荐和社区交流等功能。

## 功能特点

- **手机优先设计**: 专门为手机浏览设计的响应式布局
- **摄影主题**: 针对摄影爱好者的内容结构
- **作品展示**: 摄影作品展示区域
- **摄影教程**: 手机摄影技巧和教程
- **器材推荐**: 摄影器材和配件推荐
- **社区交流**: 摄影爱好者交流社区

## 手机优化技术

1. **触摸友好的UI**: 大按钮、滑动支持、触摸响应
2. **响应式布局**: 自动适应各种手机屏幕尺寸
3. **性能优化**: 快速加载，适合手机网络
4. **深色模式**: 自动适应手机深色模式设置
5. **手机导航**: 汉堡菜单，适合手机操作
6. **手机预览**: 手机界面预览组件

## 网站内容结构

1. **首页**: 欢迎页面和手机预览
2. **作品展示**: 摄影作品展示区
3. **摄影教程**: 摄影技巧和教程
4. **器材推荐**: 摄影器材推荐区
5. **社区交流**: 摄影爱好者交流区

## 部署说明

这个网站部署在 GitHub Pages 上，提供免费的 HTTPS 托管。

## 使用方法

在任何手机上打开网站即可体验优化的设计。网站兼容：

- iOS Safari
- Android Chrome
- Mobile Firefox
- 所有现代手机浏览器

## 开发修改

如需修改网站：
1. 编辑 `index.html` 修改内容
2. 编辑 `styles.css` 修改样式
3. 编辑 `script.js` 修改交互功能
